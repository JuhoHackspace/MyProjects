Important: This coderunner only provides the standard output of the code sent to the server. This means that only lines printed out with "console.log" will be visible. Using console.log() in this editor, does not automatically add a newline character in the print statement. Use "\n" to achieve a line break. This is due to the coderunner not accepting methods such as "process.stdout.write()", that do not include the newline character by default. The coderunner was also implemented in a way, that it does not recognize keywords such as "document", as they are not defined in the context where the code is executed(server). To use a more comprehensive HTML/JavaScript coderunner, where you can test JavaScript in a more complete environment and manipulate HTML content, visit the Learning tab and follow instructions.
